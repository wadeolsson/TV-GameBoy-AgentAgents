# TV-GameBoy

Template launcher for student-made games. The launcher shows a menu of games and starts the selected game.

## Quick start
- Install Java 17 and Gradle.
- Run: `gradle :launcher:run`

## Project layout
- `shared/` shared interfaces used by the launcher and games.
- `launcher/` menu UI.
- `games/` one module per game.
